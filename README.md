# Number Guess Challenge

A simple and interactive number guessing game built with Python and Tkinter.

## Description

Number Guess Challenge is a fun game where players try to guess a randomly generated number between 1 and 50. The game provides hints and tracks your score across 5 attempts.

## Features

- **Random Number Generation**: The computer secretly picks a number between 1-50
- **Smart Hints**: Get feedback on whether your guess is too high or too low
- **Score Tracking**: Start with 5 points and lose 1 point for each incorrect guess
- **Input Validation**: Handles invalid inputs gracefully
- **User-Friendly Interface**: Clean and intuitive GUI with colorful elements

## How to Play

1. Run the game
2. Enter your guess in the input field
